# アプリの概要
フェンシングのローカル大会を検索・公開を目的とする Web アプリである。
「大会を探す」「大会を登録する」の 2 つのページからなる。
データベースには FireStore を使う。

# 「大会を探す」ページ
以下のカラムからなるテーブルが表示される。
- 大会名
- 会場
- 種目
- 性別
- 年齢区分
- 大会開始日（ソート可）
- 大会終了日（ソート可）
- 申込締切日（ソート可）
- URL
- 備考
- 大会情報の登録日（ソート可）
- 削除（登録時に設定した削除コードの入力が必要）

#　「大会を登録する」ページ
以下の項目を入力し、「登録ボタン」を押下すると登録できる。その際、削除コードが発行される。
- 大会名（必須項目、100 文字まで）
- 会場（必須項目、100 文字まで）
- 種目（必須項目、フルーレ・エペ・サーブル・シッティングから該当する項目をすべて選択）
- 性別（必須項目、女性・男性からから該当する項目を全て選択）
- 年齢区分（必須項目、未就学児・小学生・中学生・高校生・大学生・大人・ベテランからから該当する項目を全て選択）
- 大会開始日（必須項目）
- 大会終了日（必須項目）
- 申込締切日（必須項目）
- URL（任意項目）
- 備考（任意項目）

# コーディング規約
- ロジックはカスタムフックにできるだけ閉じ込めること
- 複数回使う型やコンポーネントは共通化すること
